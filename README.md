# Serene Escapes 

A responsive resort and travel website where users can explore resorts, view offers, book trips, and contact the team.

# Project Overview

Serene Escapes is a fully static HTML/CSS website built to help users browse resort options, explore packages, and make bookings easily online. The project showcases a clean layout, intuitive navigation, and responsive design suitable for desktop and mobile screens. ([GitHub][1])

# Pages & Navigation

* **Home (index.html):** Welcome page with navigation and highlights of features. ([GitHub][1])
* **About Us (AboutUs.html):** Details about the project and Serene Escapes platform. ([GitHub][1])
* **Resorts (resorts.html):** Displays available resorts and properties. ([GitHub][1])
* **Offers (offers.html):** Showcases current packages and deals. ([GitHub][1])
* **Booking (booking.html):** Allows users to book selected packages or resorts. ([GitHub][1])
* **Contact (contact.html):** Contact form to reach the team. ([GitHub][1])
* **Login & Sign Up:** Pages for user login and registration (static templates). ([GitHub][1])

# Technologies Used

* **HTML5** – Structure and content of the pages. ([GitHub][1])
* **CSS3** – Styling and responsive layout. ([GitHub][1])
* No frameworks or JavaScript libraries — pure frontend. ([GitHub][1])

# How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/hurairasarfrazuet/group4-sereneescapes.git
   ```

2. Open `index.html` in your browser to view the Home page.

3. Navigate between pages using the menu. ([GitHub][1])
